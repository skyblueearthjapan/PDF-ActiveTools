# PDF Active Tools

ブラウザで動作するPDF結合・分割ツールです。サーバーレスで、すべての処理をクライアントサイドで実行します。

## 🚀 特徴

- **完全にブラウザ内で動作** - サーバーにファイルをアップロードせず、プライバシーを保護
- **直感的なUI** - ドラッグ&ドロップで簡単にPDFを操作
- **豊富な機能**
  - 複数PDFの結合（Merge）
  - PDFの分割（Split）
  - ページの回転（90度単位）
  - ページの並び替え
  - ページ単位での選択・削除

## 📋 必要な環境

- Node.js 16.x 以上
- npm または yarn

## 🛠️ セットアップ

### 1. リポジトリのクローン

```bash
git clone https://github.com/yourusername/PDF-ActiveTools.git
cd PDF-ActiveTools
```

### 2. 依存関係のインストール

```bash
npm install
```

### 3. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

## 📦 ビルド

本番用ビルドを作成するには：

```bash
npm run build
```

ビルド結果は `dist/` ディレクトリに出力されます。

ビルド結果をローカルでプレビューするには：

```bash
npm run preview
```

## 🎯 使い方

### Merge（結合）モード

1. 左側のエリアにPDFファイルをドラッグ&ドロップ、またはクリックしてアップロード
2. アップロードされたPDFのページサムネイルが表示されます
3. 結合したいページを「出力タイムライン」にドラッグ&ドロップ
4. ページの順序を変更したい場合は、タイムライン内でドラッグして並び替え
5. 必要に応じて「回転」ボタンでページを90度ずつ回転
6. 「PDFを結合してダウンロード」ボタンをクリック

### Split（分割）モード

1. 左側のリストから分割したいPDFファイルを選択
2. 分割方法を選択：
   - **すべてのページを1ページずつ分割** - 各ページを個別のPDFに
   - **ページ範囲で分割** - 範囲指定（例: 1-3, 4-10）
   - **選択したページのみで1つのPDFを作成** - 必要なページだけを抽出
3. 「PDFを分割してダウンロード」ボタンをクリック

## 🔧 使用技術

- **React** - UIフレームワーク
- **TypeScript** - 型安全な開発
- **Vite** - 高速ビルドツール
- **pdf-lib** - PDF操作ライブラリ

## 🌐 GitHub Pagesへのデプロイ

1. `vite.config.ts` の `base` オプションをリポジトリ名に設定：

```typescript
export default defineConfig({
  plugins: [react()],
  base: '/PDF-ActiveTools/', // リポジトリ名
  build: {
    outDir: 'dist',
  }
})
```

2. ビルドとデプロイ：

```bash
npm run build
# dist ディレクトリの内容を gh-pages ブランチにプッシュ
```

または、GitHub Actionsを使用した自動デプロイも可能です。

## ⚠️ 制限事項

- 非常に大きなPDFファイル（100MB以上）は処理に時間がかかる場合があります
- ブラウザのメモリ制限により、極端に大きなファイルは処理できない可能性があります
- サムネイル生成は簡易的なものです（実際のPDFレンダリングは行っていません）
